# Text to PDF Converter

A simple, client-side tool to convert plain text into PDF files. Runs entirely in your browser — no data is uploaded anywhere.

## Features

- Convert text to PDF instantly
- Customizable font size
- Choose between Helvetica, Times New Roman, or Courier
- 100% client-side (privacy-friendly)
- Free and open source

## Usage

1. Open `index.html` in your browser
2. Type or paste text
3. Click "Download PDF"

- HTML & CSS
